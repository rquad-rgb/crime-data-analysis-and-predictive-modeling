# Crime Data Analysis & Predictive Modeling

## Data
The dataset used in this project contains crime records from Los Angeles (2020–Present).  
It was sourced for academic analysis and may require preprocessing due to missing values and inconsistencies.

## Overview
This project explores crime patterns in Los Angeles and builds a predictive model to determine arrest outcomes.  
It follows a full data science workflow, including data cleaning, exploratory data analysis, and statistical modeling.

## Objectives
- Analyze crime trends over time  
- Identify patterns across victim demographics  
- Clean and preprocess real-world crime data  
- Build a classification model for arrest prediction  
- Evaluate model performance on unseen data  

## Methods Used
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Data Visualization  
- Logistic Regression (Binary Classification)  
- Train/Test Split for Model Validation  

## Key Findings
- Crime rates increased from 2021 to 2022, with a slight decline afterward (likely due to incomplete recent data)  
- Certain victim demographic groups show higher reported crime counts, though this may reflect population distributions or reporting bias  
- Logistic regression identified victim age, sex, and location as meaningful predictors of arrest outcomes  
- Proper handling of missing data and categorical variables was essential for model performance
